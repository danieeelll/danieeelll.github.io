Page layout is based on the checkerboard structure: there are 3 vertical sections and each of them has its own horizontal plane with more sections. The movement through the page could be controlled in these several ways.

1. Please dont forget to try the "sensitive mouse wheel control" I mentioned above. It is simply perfect. But for some effect is better to use the standart scroll bar for vertical movement - this is the case of the Parallax effect for example.

❯
❮

2. Especially in order to ensure a comfortable movement in the horizontal direction there are the side arrows control elements here. In the case that next page is available the arrow will be visible. If you are at the begining or end of the horizontal section the arrow will automaticaly not be displayed. And vice versa. Small pictogram will explain it best.

3. And finally you could anytime use the fixed menu bar on the top of the displayed area. You reach every corner of this page in very comfortable way when you use it.

Watter ripples effect

A nice "watter surface ripple" effect builded on jQuery framework by Pim Schreurs / Sirxemic from Netherlands. Many thanks to him this good work - I like it a lot. There is the link on the demo page for this effect and link on his GitHub here: http://sirxemic.github.io/jquery.ripples/

Please note this is one from the effect that are using a lot of computing power. So if you feel that your device performance is going down the hill, try to turn this effect off.

Parralax background effect

Parallax is a difference in the apparent position of objects in order by the point of view from which they are observed.

The imitation of this real phenomenon is an great opportunity to achieve a look of spatial depth. You may look how it works here on this page and compare with the standart 2D design.

This solution is based on pure CSS witch is using the transformation on the Z-axis with 3D perspective property.

Despite it is a pure CSS solution, that manipulation with more elements in 3D perspective is really not very gentle to the system resources. That is why some other effect will be turned off (watter surface ripple) if parallax is started. (But anytime you can start it all together if you want. It is depending on the hardware, system load, free system resources... Have to try. )

Bubble help and page guide

You may switch the displaying of the bubble help on some page elements anytime. "Page engine room" in the setting section will be still available.

If the "initial page guide sequence" is still running, please turn it off by it own switch in the left top corner of the page. When the sequence goes to finish properly, it will destroy / disable it self automatically.

Please note, that in order to a visual ilustration of some features some bubbles will still stay active here in this setting section or in the setting panel.

Glass effect on background

Will turn on this interesting background glass effect on whole page for 60 second.

Only the "setting section - page engine room" has started this effect on default. Other sections have not "glass background" turned on. You may change it for 60 second.

(If you want watch and compare the parallax transition for example...)
